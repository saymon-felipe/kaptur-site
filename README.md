## 📌 Sobre o Kaptur

O Kaptur é uma extensão de navegador focada em produtividade para criação de conteúdo em vídeo. Desenhado para ser ágil e intuitivo, ele cobre todo o fluxo de produção audiovisual direto no navegador, desde a captura da tela até o momento da publicação.

## 🚀 Como Usar: O Fluxo de Trabalho

A plataforma é dividida em três etapas principais para atender desde necessidades rápidas até produções mais elaboradas:

### 1. 🎥 Gravação
Inicie a captura com apenas um clique. O Kaptur permite gravar sua tela inteira, janelas de aplicativos específicos ou abas do navegador, com suporte simultâneo à captação do seu microfone e áudio do sistema. Ideal para tutoriais, feedbacks visuais e apresentações.

### 2. ✂️ Edição Simples
Para ajustes rápidos logo após a gravação. O módulo de edição simples permite:
- Cortar o início e o fim do vídeo (Trim).
- Remover trechos indesejados no meio da gravação.
- Revisar o material de forma ágil antes de salvar ou compartilhar.

### 3. 🛠️ Edição Pesada
Para conteúdos que exigem um acabamento profissional. Este módulo oferece ferramentas avançadas para manipular o vídeo gravado, permitindo uma pós-produção mais detalhada sem a necessidade de exportar o arquivo para softwares externos pesados.
